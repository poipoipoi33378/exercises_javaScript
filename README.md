# JavaScript課題
## 提出課題のフォルダ構成について
課題毎のフォルダ内に実行用HTMLファイル・ソースコードフォルダ（src）・テスト用フォルダ（spec）を作成しています。

例）課題１の場合

- 課題１
  - fizzbuss.html
    - 課題実行用HTMLファイル
    - 課題８は、テストでのみ実行確認
  - src
    - FizzBussSpec.js
    - テスト
  - spec
    - FizzBuss.js
    - 課題の回答

テストの実行
- 以下のファイルより実行可能
  - jasmine-standalone-3.2.1/SpecRunner.html
  - 課題６に関しては、テストを書かず、実装したのでテストは作っていない。
